# ShutdownControl

應用於防家人嫌浪費電及自動關機需求 for Windows on C#

## 特色

* 利用縮小視窗化程式進行自動關機

## 使用方法


```
$ git clone https://github.com/KuKuOOOOOO/ShutdownControl
```
將專案clone至Windows

```
$ ..ShutdownControl\ShutdownControl\bin\Debug\ShutdownControl.exe
```
進入路徑點選ShutdownControl.exe
